# S3 Bucket Audit Project

This project audits S3 buckets for public access and encryption status.

## Checks Performed
- Checks if S3 buckets have public access.
- Checks if S3 buckets are encrypted.

## Requirements
- Python 3.10 or higher
- Boto3 library
